# GithubLanguageCalculator
Calculates the languages inside a GitHub profile.

# Usage
``` bash
git clone [REPO_LINK]
cd GithubLanguageCalculator
npm install
node index.js [GITHUB_USER_NAME]
# For example: node index.js nabilridhwan
```

## Possible Output:
Example:
``` json
{
    "JavaScript": 3,
    "Python": 1,
    "Java": 1,
    "Total": 5
}
```
